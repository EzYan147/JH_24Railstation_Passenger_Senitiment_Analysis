五、三大方向专题:旅客服务 · 工程建设 · 站房改造

三大管理维度声量

维度内各主题负面率

① 铁路旅客服务(提及 106,484 · 负面 23,994 · 负面率22.5%)

  • 信息服务是第一痛点:列车运行信息负面2.16万条,占全部负面77%。晚点通知滞后、检票口临时变更、停检规则认知不足(南京南“提前3分钟停检”3,113赞)、车次增减无解释(滕州东/丹阳北/定远)四大场景 → 建立“购票-候车-乘降”全链路信息推送标准 + 车次调整地方化解释机制。
  • 一线服务是最大口碑资产:工作人员服务负面率仅18%,济南西(讨论4,779条、正向59%)的导盲犬服务、应急救助屡成正面爆款 → 重点旅客服务、应急救助标准化,全线复制并主动传播。
  • 安检与进出站:德州东高峰通过能力不足、蚌埠南证件识别设备故障直接造成误车纠纷 → 设备可靠性排查 + 高峰人工兜底通道。
  • 商业价格与客流安全:高价与纠纷事件(南京南)易被高赞放大 → 明码标价巡查、纠纷快速处置;站台安全防护排查 + 大客流分级响应。

② 工程建设(提及 31,157 · 负面 8,555 · 负面率27.5%)

  • 选址与覆盖:宿州东(“本地人叫灵璧西站”)、苏州北(“又偏又远”)的可达性抱怨持续,蚌埠南暴露怀远等百万人口县域覆盖缺口 → 新站论证纳入“门到门时间”,既有站以快速接驳补救。
  • 站前交通工程:无锡东网约车占道、昆山南南广场拥挤+单行窄路、北京南出租车蓄车、德州东落客拥堵 → 投入小、感知大,优先改造。
  • 扩建工程舆情:苏州北480亿扩建、泰安扩容呼声、沧州西“夹心”布局争议 → 工程进展定期公开,把期待转化为正面声量。
  • 命名与标识:定远(负面率70%话题源)、滁州北更名、常州北/常州站混淆 → 地图/票面/站房三处统一专项。

③ 站房改造(提及 47,858 · 负面 18,510 · 负面率38.7%,三维度最高)

  • 温控与隔热:北京南“冬冷夏热”(3,066赞)是玻璃幕墙站房共性短板 → 纳入站房大修专项,优先高架候车层。
  • 照明与空间氛围:天津西“夜间乌漆麻黑”(5,485赞)证明照明是低成本高感知项 → 对夜间到发量大的车站做照明专项评估。
  • 候车设施:泰安按摩椅侵占座位(5,316赞)、德州东/丹阳北候车舒适负面率高 → 座椅数量与商业设施配比设红线,卫生间对标机场。
  • 动线与标识:南京南“容易迷路就失败”(7,374赞)、徐州东换乘绕行 → 以“全链路找路实验”系统重排标识。
  • 无障碍与适老化:滁州北天桥无电梯是全线最具体整改项 → 借此对24站无障碍设施拉网排查。
  • 形象与可用性平衡:站房设计形象主题负面率36%,“好看但不好用” → 新改建项目验收纳入可用性指标(找路时间、座位率、温控达标率)。

结语
